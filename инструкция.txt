План действий, чтобы получить полностью рабочее окружение:
0. Устанавливаешь себе на машину Ansible
1. Создаешь три виртуалки (на убунте)
2. В файлике hosts заменяешь айпишники на айпишники, которые будут у виртуалок
3. Пишешь в консоли команду ansible-playbook -i hosts install_docker.yml

Все должно отработать. Чтобы проверить, что все отработало штатно нужно сделать:
1. Перейти по ссылке http://<nginx-ip>, где вместо <nginx-ip> нужно вставить ip машины, на которой запущен nginx (то есть самой верхней из файлика hosts). Тут ты увидишь статическую страницу, полученную из nginx
2. Перейти по ссылке http://<nginx-ip>/test/index.jsp. Тут ты увидишь сообщениие, полученную из apache tomcat. Это значит что работает разделение трафика на статику и динамику
3. Пообновлять страницу несколько раз. Поскольку nginx выступает в качестве reverse-proxy между серверами tomcat, то и отправлять запросы он будет каждый раз на разные сервера tomcat. Поэтому ты постоянно будешь видеть приветственное сообщение с отличающимся IP адресом.
